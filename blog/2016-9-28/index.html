0.7 Release¶

We continue our monthly release cycle with our September release

Thanks to Bruno Moura’s work, Docker volume plugin for vSphere now supports selection of file system format when creating a new docker volume.

Ability to set a volume read-only is now supported by the plugin. The admin cli allows the admin to set a volume to be read only.

Coming up¶

One of the major features in the pipeline is multi-tenancy and vCenter integration. More about that soon..
